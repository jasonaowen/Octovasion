# GIP - Game In Progress

A game for the Recurse Center game jam!

## Building

This project uses CMake.

```sh
$ mkdir build
$ cd build
$ cmake -G "Unix Makefiles" ..
$ make catch
$ make && ./Test && ./GIP
```
